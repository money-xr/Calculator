# 🧮 Calculator

A simple calculator built using **HTML, CSS, and JavaScript**. This project was created by combining my own understanding of frontend development with AI assistance for design ideas and implementation.

## ✨ Features

- Basic arithmetic operations (+, −, ×, ÷, %)
- Decimal number support
- Backspace and All Clear (AC)
- Expression display
- Responsive UI
- Keyboard support (Esc to clear)

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🚀 Run Locally

1. Download or clone the repository.
2. Open `index.html` in your browser.

## 📁 Files

- `index.html` – Structure
- `style.css` – Styling
- `script.js` – Calculator logic
